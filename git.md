####space name
    Workspace：工作区
    Index / Stage：暂存区
    Repository：仓库区（或本地仓库）
    Remote：远程仓库
    

###branch
######新建并切换分支
    git checkout -b branch1 == git branch branch1 + git checkout branch1
    
######list all remote branch
    git branch -r
    
######list all local and remote branch
    git branch -a
